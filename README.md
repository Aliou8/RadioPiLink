# RadioPiLink

RadioPiLink est un projet IoT permettant la transmission de données radio depuis un microcontrôleur STM32WLE5 vers un Raspberry Pi, puis leur relais sur un réseau local pour traitement et visualisation.

## Table des matières
- [Fonctionnalités](#fonctionnalités)
- [Architecture](#architecture)
- [Licence](#licence)

## Fonctionnalités
- Réception radio sur STM32WLE5
- Passerelle vers Raspberry Pi
- Routage des données vers le réseau local
- Points d'extension pour traitement et visualisation
